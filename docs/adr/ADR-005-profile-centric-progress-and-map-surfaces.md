# ADR-005 Profile-Centric Progress and Map Surfaces

## Context

Progress and learning map are secondary learner reflections, not top-level navigation goals on every screen.

## Decision

Place progress insights and learning map inside profile as tabs. Keep compatibility routes as redirects.

## Consequences

- clearer primary navigation
- one learner-hub surface for reflection and analysis
- older links can continue working via redirects

## Non-goals

- removing the underlying progress or map functionality
